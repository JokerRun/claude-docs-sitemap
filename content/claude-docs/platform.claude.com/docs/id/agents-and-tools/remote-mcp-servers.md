# Server MCP jarak jauh

Server MCP jarak jauh yang dapat dihubungkan oleh pengembang melalui API konektor MCP Anthropic.

---

Beberapa perusahaan telah menerapkan server MCP jarak jauh yang dapat dihubungkan oleh pengembang melalui API konektor MCP Anthropic. Server-server ini memperluas kemampuan yang tersedia bagi pengembang dan pengguna akhir dengan menyediakan akses jarak jauh ke berbagai layanan dan alat melalui protokol MCP.

<Note>
    Server MCP jarak jauh yang tercantum di bawah ini adalah layanan pihak ketiga yang dirancang untuk bekerja dengan API Claude. Server-server ini
    tidak dimiliki, dioperasikan, atau didukung oleh Anthropic. Pengguna hanya boleh terhubung ke server MCP jarak jauh yang mereka percayai dan
    harus meninjau praktik keamanan dan ketentuan setiap server sebelum terhubung.
</Note>

## Menghubungkan ke server MCP jarak jauh

Untuk terhubung ke server MCP jarak jauh:

1. Tinjau dokumentasi untuk server spesifik yang ingin Anda gunakan.
2. Pastikan Anda memiliki kredensial autentikasi yang diperlukan.
3. Ikuti instruksi koneksi khusus server yang disediakan oleh setiap perusahaan.

Untuk informasi lebih lanjut tentang menggunakan server MCP jarak jauh dengan API Claude, lihat [dokumentasi konektor MCP](/docs/id/agents-and-tools/mcp-connector).
