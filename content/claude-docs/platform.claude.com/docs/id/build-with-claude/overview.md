# Ikhtisar fitur

Jelajahi fitur dan kemampuan lanjutan Claude.

---

Permukaan API Claude diorganisasikan ke dalam lima area:

* **Kemampuan model:** Mengontrol bagaimana Claude bernalar dan memformat respons.
* **Alat:** Memungkinkan Claude mengambil tindakan di web atau di lingkungan Anda.
* **Infrastruktur alat:** Menangani penemuan dan orkestrasi dalam skala besar.
* **Manajemen konteks:** Menjaga sesi yang berjalan lama tetap efisien.
* **File dan aset:** Mengelola dokumen dan data yang Anda berikan kepada Claude.

Jika Anda baru memulai, mulailah dengan [kemampuan model](#model-capabilities) dan [alat](#tools). Kembali ke bagian lainnya ketika Anda siap untuk mengoptimalkan biaya, latensi, atau skala.

Untuk administrasi dan tata kelola, lihat [Admin API](/docs/id/manage-claude/admin-api), [Usage and Cost API](/docs/id/manage-claude/usage-cost-api), dan [Compliance API](/docs/id/manage-claude/compliance-api).

## Ketersediaan fitur

Fitur pada Claude Platform diberi salah satu klasifikasi ketersediaan berikut per platform (ditampilkan di kolom Ketersediaan pada setiap tabel berikut). Tidak semua fitur melewati setiap tahap. Sebuah fitur dapat masuk pada klasifikasi apa pun dan dapat melewati beberapa tahap.

| Klasifikasi                   | Deskripsi                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Beta**                      | Fitur pratinjau yang digunakan untuk mengumpulkan umpan balik dan melakukan iterasi pada kasus penggunaan yang kurang matang. Ketersediaan mungkin terbatas, termasuk melalui persyaratan pendaftaran atau daftar tunggu, dan mungkin tidak diumumkan secara publik. Fitur dapat berubah secara signifikan atau dihentikan berdasarkan umpan balik. Tidak dijamin untuk penggunaan produksi yang berkelanjutan. Perubahan yang merusak (breaking changes) mungkin terjadi dengan pemberitahuan, dan beberapa batasan khusus platform mungkin berlaku. Fitur beta pada Claude API dan [Claude Platform on AWS](/docs/id/build-with-claude/claude-platform-on-aws) memiliki [header beta](/docs/id/api/beta-headers). |
| **Tersedia secara umum (GA)** | Fitur stabil, didukung penuh, dan direkomendasikan untuk penggunaan produksi. Seharusnya tidak memiliki header beta atau indikator lain bahwa fitur berada dalam status pratinjau. Dicakup oleh jaminan [versioning](/docs/id/api/versioning) API standar.                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **Deprecated**                | Fitur masih berfungsi tetapi tidak lagi direkomendasikan. Jalur migrasi dan garis waktu penghapusan disediakan.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Retired**                   | Fitur tidak lagi tersedia.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

**Label platform:** Claude API (pihak pertama Anthropic) · [Bedrock](/docs/id/build-with-claude/claude-in-amazon-bedrock) (dioperasikan AWS) · [Claude Platform on AWS](/docs/id/build-with-claude/claude-platform-on-aws) (dioperasikan Anthropic di AWS) · [Google Cloud](/docs/id/build-with-claude/claude-on-vertex-ai) (dioperasikan Google) · [Microsoft Foundry](/docs/id/build-with-claude/claude-in-microsoft-foundry) (dioperasikan Anthropic di Azure)

## Kemampuan model

Cara untuk mengarahkan Claude dan output langsung Claude, termasuk format respons, kedalaman penalaran, dan modalitas input.

<Tip>
  Anda dapat menemukan kemampuan apa saja yang didukung oleh sebuah model secara terprogram. [Models API](/docs/id/api/models/list) mengembalikan `max_input_tokens`, `max_tokens`, dan objek `capabilities` untuk setiap model yang tersedia.
</Tip>

Kolom ZDR menunjukkan apakah sebuah fitur tersedia di bawah pengaturan "Zero Data Retention" (Retensi Data Nol), atau ZDR. Untuk sebagian besar fitur, ini hanya bergantung pada apa yang disimpan oleh mekanisme fitur tersebut; untuk fitur yang terikat pada model tertentu, ketersediaan ZDR tingkat model juga berlaku. Lihat [Persyaratan retensi data khusus model](/docs/id/manage-claude/api-and-data-retention#model-specific-data-retention-requirements).
